# Status Code

Part of the Fridge Combo project. A lightning-fast reference tool for HTTP status codes.

## 1 Tool / 1 Action

Decode the protocol. **Status Code** translates 3-digit HTTP response numbers into human-readable insights instantly.

## Philosophy
* **1 Tool, 1 Action.**
* **Developer-Focused:** Stripped-down UI for professionals.
* **No Login / No Ads / No Tracking**
* **Browser-only:** Works instantly on any device.

## How to Use
1. **Type:** Enter a 3-digit status code (e.g., 404, 500, 418).
2. **Instant Result:** Read the meaning and suggested action immediately.
3. **Explore:** Discover the hidden "Teapot" and other protocol secrets.
